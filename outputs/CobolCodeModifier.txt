               MOVE 8 TO LS-RETURN-CODE
               PERFORM 3000-TERMINATE
               GOBACK
           END-IF
      *-- Change: Open alarm log file for alarm persistence
           OPEN EXTEND ALARM-FILE
           IF WS-ALARM-STATUS NOT = '00'

               GOBACK

               MOVE 8 TO LS-RETURN-CODE

      *-- Change: Write alarm record if type is ALRM (UDF-level alarm)

                   DISPLAY 'Error writing alarm record: ' WS-ALARM-STATUS
                   MOVE 8 TO LS-RETURN-CODE
               END-IF
           END-IF
           .
           
       3000-TERMINATE.
           CLOSE AUDIT-FILE
      *-- Change: Close alarm file
           CLOSE ALARM-FILE
